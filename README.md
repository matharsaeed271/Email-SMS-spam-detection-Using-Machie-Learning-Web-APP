# Email-SMS-spam-detection-Using-Machie-Learning-Web-APP

#### Now scale and classify your thoughts with the cutting edge web app
This web application is designed to detect whether a given email or SMS message is spam or not using machine learning techniques. The system analyzes the input text, processes it through a trained model, and classifies it as either “Spam” or “Not Spam.”
This project demonstrates the practical use of machine learning in cybersecurity and communication filtering, helping users avoid unwanted or harmful messages.
The web application can be accessed using the following link:
👉   https://spamsdetect.streamlit.app/
